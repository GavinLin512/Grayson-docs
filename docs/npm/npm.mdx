## npm install 報錯可能原因
> 由於套件過舊，**node** 找不到其相對應的版本

* 解決方法：由於 **npm** 套件仰賴 **node** 版本，此時可以先看看錯誤碼去尋找該套件release的版本，再根據需求對 **node**
做升級或降版。
